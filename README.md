# UI编辑器辅助

### 目的
    提升负责UI的美术小伙伴的工作效率


### 功能
    针对非WorldSpace(世界空间)的Canvas(舞台控件)中的UI元素提供以下功能
    * 可自定义显示网格
    * 可控的网格吸附开关
    * 显示当前UI元素的重绘估值(在UI布局时提供一个参考)
